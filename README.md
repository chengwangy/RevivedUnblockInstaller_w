### 如何安装？

1. 安装 [BetterNCM](https://github.com/MicroCBer/BetterNCM) 插件管理器
2. 点击 右上角出现的云音乐按钮
3. 安装 RevivedSource
4. 重载后，安装 RevivedUnblockInstaller

### 如何使用？

1. 进入 插件设置（点击 RevivedUnblockInstaller）
2. 点击 “已禁用” 按钮来启用
3. 选择你需要的版本（国内用户推荐 `-ghproxy` 的版本），点击应用
4. 按流程操作即可
